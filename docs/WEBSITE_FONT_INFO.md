## üìà Performance Considerations

### Loading Strategy
```css
display=swap
```
**Benefit:** Shows fallback font immediately, swaps when Google Font loads (prevents invisible text)

### Font Weights Loaded
Only necessary weights are loaded:
- 300, 400, 600, 700, 900 (not 100, 200, 500, 800)
- Reduces file size
- Faster page load

### Subset (if needed)
Could be optimized further with:
```
&text=YourSpecificCharacters
```
But current setup is fine for performance.
